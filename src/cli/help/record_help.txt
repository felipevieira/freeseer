                        ##### FreeSeer Help #####

		           >>> Record Tool <<<


Description> This CLI tool allows the user to perform record operations through the CLI. The user can perform all task suported by FreeSeer by handling one of the available modes described below.

Available Modes> These are the record tasks available
		
	# 'record': it records the desktop to a file called 'default.ogg' on the video folder
	# 'record -p #': it records the presentation with the specified id to its respective file on the user video folder
	# 'record -o <file>': it records the desktop to the specified file.
